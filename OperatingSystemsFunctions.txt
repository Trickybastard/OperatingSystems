
Device management- Manages all device communications with the help of drivers. It keeps track of all devices currently running and on communicating with the machine, the program that does this is called the "I/O controller." Device management also decides which process gets control of device and how much resources and time it gets.

File manager- File manager is a default file browser typically organized into directories for easy naviagtion and use. These directors will usually contain everything from files to applications. The file manager helps to keep track of information.
